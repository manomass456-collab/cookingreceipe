# CulinaShare 🍳

A modern, responsive cooking recipe website built with vanilla web technologies. Browse delicious meals, filter by category, and keep your kitchen organized.



## 🚀 Features
* **Dynamic Recipe Gallery:** Browse a wide collection of recipes with high-quality imagery.
* **Real-time Search:** Filter recipes by name or ingredients as you type.
* **Category Filtering:** Quickly sort through breakfast, lunch, dinner, and snacks.
* **Interactive Recipe Cards:** Click to view detailed instructions, prep time, and serving sizes.
* **Favorites List:** Save your must-try recipes (powered by LocalStorage).
* **Fully Responsive:** Optimized for everything from mobile screens to large desktops.

## 🛠️ Tech Stack
* **HTML5:** Semantic structure for better SEO and accessibility.
* **CSS3:** Custom styling with Flexbox and Grid layouts for a modern look.
* **JavaScript (ES6+):** Handles dynamic content rendering, search logic, and data persistence.
